# Giga 🗿

Giga is a Flutter-based self-improvement tracking application designed to help users build discipline and track their progress. It features a milestone-based reward system where users earn badges and unlock the "Giga Chad" status as they maintain their streak.

## 🚀 Features

  * **Streak Tracker:** Monitor exactly how many days you have stayed committed to your goals.
  * **Badge System:** Earn unique badges for reaching specific milestones (7 days, 30 days, 90 days, etc.).
  * **Giga Chad Status:** Reach the ultimate tier by maintaining long-term discipline.
  * **History & Personal Bests:** Keep a record of your longest streaks to stay motivated and beat your past self.
  * **Built with Flutter:** Smooth UI and cross-platform performance.

## 📱 Getting Started

This project is a starting point for a Flutter application. To get a local copy up and running, follow these simple steps:

1.  **Prerequisites:** Ensure you have the [Flutter SDK](https://docs.flutter.dev/get-started/install) installed.
2.  **Clone the repo:**
    ```bash
    git clone https://github.com/yourusername/giga.git
    ```
3.  **Install dependencies:**
    ```bash
    flutter pub get
    ```
4.  **Run the app:**
    ```bash
    flutter run
    ```
